使用Windows系统计算机和无线连接器进行连接投屏

确保教室内的无线连接器正常，并且电脑具有至少一个USB-A接口

将无线连接器插入电脑的USB-A进行以下操作：

Windows投屏
  1. 双击桌面图标此设备
  2. 在弹出的窗口中双击设备和驱动
  3. 双击新增的驱动并跟随指南安装驱动
  4. 成功安装后点击驱动图标运行软件
  5. 按压无线连接器上的圆形按键
